When enabled, all users share a combined view of the company list. Each user can still see their own company details; other users' private notes are hidden. Application details are never shared across user accounts. Only meaningful when login is enabled.
About Reminders

The built-in reminder system uses a background scheduler (APScheduler) that runs inside the Flask server process. It checks for pending applications every hour and adds a message to the Inbox when the threshold is exceeded.

No external services, emails, or browser extensions are required.

About Ollama Integration

Ollama lets you run large language models locally on your own machine. Once configured, the AI Assistant can help you draft cover letters, summarise job descriptions, or suggest improvements to your application notes — all without sending data to any external cloud service.

Quick start:

  1. Install Ollama from ollama.com
  2. Run ollama pull llama3 (or another model)
  3. Enter the server URL above (default: http://localhost:11434)
  4. Click Test to verify the connection

When enabled, an Ask AI button will appear on the application detail page to let you ask questions about that application.

How to Update

This app is open-source and hosted on GitHub. To update to the latest version:

  1. Back up your data first — go to Export → Download Full Database to save a copy of your data before making any changes.
  2. Stop the running server
  3. Run git pull origin main in the project directory
  4. Install any new dependencies: pip install -r requirements.txt
  5. Restart the server

If you are using Docker, rebuild the image after pulling: docker compose up --build -d

No Git? See the manual update guide for instructions on downloading a release archive or pulling a Docker image directly.
