{# Quick-tip box on overview page #}
Debug Tips
  • Application not showing up? Browse applications and check the user_id column — make sure it matches your account ID shown in the users table.
  • Login locked out? Use the SQL console and run:
    UPDATE settings SET value='0' WHERE key='login_enabled'
  • AI settings not saving? Check user_ai_settings — rows are keyed by user_id. If no row exists for your user, save your settings in the AI tab first.
  • Status order wrong? Browse statuses and edit the sort_order values directly here.
  • Onboarding stuck? Run: UPDATE settings SET value='1' WHERE key='onboarding_complete'
